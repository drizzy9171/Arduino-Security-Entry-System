(AI Written)# Arduino Password Entry System

## Description
A password entry system using a 4x4 keypad with LED indicators for success/failure.

## Components
- Arduino Uno
- 4x4 Keypad
- Green LED
- Red LED
- 2x 220Ω Resistors

## How to Use
1. Enter 4-digit password using keypad
2. Green LED = Access Granted
3. Red LED = Access Denied
4. Default password: 1234
